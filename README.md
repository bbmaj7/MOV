# MOV
MOV merupakan sebuah project dari online class BuildWithAngga.com dengan menggunakan Kotlin sebagai front-end dan menggunakan Firebase sebagai back-end

## Fitur
1. Splashscreen
2. Onboarding
3. SignIn dan SignUp
4. Home
5. Movie List
6. Movie Detail
7. Order
8. E-Wallet
9. Success Order Notification

## Tools
1. Android Studio
2. Adobe XD
3. Github
4. Android SDK
5. Firebase
